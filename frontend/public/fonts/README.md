# OpenDyslexic Fonts

These fonts are part of the dyslexia mode feature.

Copy the font files from `dyslexia mode/fonts/` to this directory:
- OpenDyslexic-Regular.otf
- OpenDyslexic-Bold.otf
- OpenDyslexic-Italic.otf
- OpenDyslexic-BoldItalic.otf

The fonts will be loaded when dyslexia mode is enabled.
